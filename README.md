# initialize project 
```
npm init -y
```

# initialize typescript
```
npx tsc --init
```

# install prisma
```
npm i prisma 
```

# initialize prisma
```
npx prisma init
```

# generate prisma client
```
npx prisma generate
```

# migrate table changes
```
npx prisma migrate dev --name init
```

# view tables and data
```
npx prisma studio
```
